ETIQUETAS SEMANTICAS ESTRUCTURALES

Es una etiqueta contenedora que no tiene semantica
header - Cabecera de un sitio web o de una seccion
main - Defina la seccion principal del documento, solo puede existir un solo por documento
Footer - Pie de pagina de un sitio web o de una seccion
nav- representa una navegacion
article - Representa un seccion autocontenido (que por si sola se explica)
aside- un contenido complementario o secundario
section - Representa una seccion de contenido generico
address - Representa una direccion
